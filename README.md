# Premium Immersive IT Company Website

This repository is structured to present the project like a serious production concept rather than a single loose demo file.

It contains a premium homepage prototype for an IT company website focused on immersive storytelling, premium motion, refined UI, and conversion-focused presentation.

## Repository Purpose

This repo is designed for:

- client review
- concept validation
- design direction presentation
- front-end quality demonstration
- deployment preview

## Included Files

- `immersive-it-homepage-demo.html`  
  Main interactive homepage concept

- `docs/PROJECT_OVERVIEW.md`  
  Explains the strategy, concept, and experience goals

- `docs/DESIGN_DIRECTION.md`  
  Captures visual language, motion style, and UX thinking

- `docs/DEPLOYMENT.md`  
  Quick deploy instructions for Netlify and GitHub

- `assets/`  
  Reserved structure for future production assets

## Quick Preview

Open `immersive-it-homepage-demo.html` directly in a browser or deploy it to Netlify.

## Netlify Deploy

1. Go to `https://netlify.com/drop`
2. Drag `immersive-it-homepage-demo.html`
3. Netlify will generate a public live URL

## Suggested Production Expansion

A full production version can evolve into:
